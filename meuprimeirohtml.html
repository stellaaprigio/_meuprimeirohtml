Conhecendo o HTML

HTML(Hyper Text Markup Language) consiste em uma linguangem de marcação padronizada para a criação de uma página web, composta por uma série de elementos que são estruturados pelo desenvolvedor, de modo que o navegador traduzirá e exibirá para o usuário a página legível e de fácil entedimento.

Com o HTML é possivél especificar para o navegador se o conteúdo que será exibido deverá ser traduzido como:


  • paragrafos;
  • lista;
  • imagens/vídeos;
  • links;
  • formulários
  • etc.

Para melhor estruturação do HTML e para tornar sua página mais dinamica para o usuário, podemos estilizá-lo via *CSS e *JavaScript.

Elementos HTMl e seus atributos

Os elementos html são basicamente as tags iniciais utilizadas para orientar o navegador o que está sendo desenvolvido,semanticamente no html.

Como por exemplo: elementos de formatação de texto

  1. Negrito: Strong, b.
  2. Itálico: i;
  3. Marca texto: mark;
  4. Sublinhado: u (underline).

Lembrando que, as tags em sua maioria se iniciam e fecham e para o fechamento utilizamos a barra (/). Algumas tags como input não possui tag de fechamento e a tag img possui auto fechamento, ou seja ela por si mesma já tem a barra no final da tag.

Já os atributos, são informções adicionais forncidas ao navegador sobre determinados elementos, esses atributos são inceridos sempre nas tags iniciais e normalmente em pares, sou seja o atributo a ser fornecido junto com o "tipo" desejado

Exemplos:

  1. Na tag input temos os atributos type para inserirmos valores de tipos textuis, númericos, barras, botões, urls, e-mails, etc
  2. Na tag img, podemos definir o src, ou seja o endereço de onde a imagem foi retirada, podendo ser o próprio navegador ou por meio de endereço web
  3. Em algumas tags podemos utilizar os atributo style por comando CSS para estilizarmos fontes, cores, etc.

    4. *Obs.: Também podemos fazer a estilização de cores no Html sem ser via CSS, utilizando as tags especificas

Para definirmos os atributos, seguimos o padrão: tag> atribui> = ""> fechamento da tag caso tenha fechamento

Listas ordenadas e não ordenadas

Para a criação de listas no html, utilizamos as tags ol para listas ordenadas que se caracteriza por meio de números (1,2,3,...) e ul que são as listas não ordenadas que são caracterizadas por marcadores (*, °, ...) variando de acordo com o navegador utilizado.

Exemplo:

Lista ordenada:

  1. Banana
  2. Morango
  3. Maça
Lista não ordenada:
  • Leite condensado
  • Chocolate
  • Creme de leite

Utilização de links

Para inserirmos links na nossa página web, seja para navegabilidade, mostrar o site de pesquisa ou indicação de outras páginas web, utilizamos a tag a, que significa ancora, e o atributo href que é onde o link ou id (identificador de elemento), caso seja para abrir ou guiar para algo dentro do próprio site, será inserido.

Quando criamos uma página e para facilitar a navegação do usuário, utilizamos target blank, que irá abrir o link em outra aba, ou self que abrirá na mesma aba.
